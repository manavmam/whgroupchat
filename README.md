**WH Group Chat Mod**

A Minecraft mod that introduces text group chat functionality using various commands.

**Tech Stack**

Language: Java

Mod Loader and Version: Forge 1.20.1

Build System: Gradle

**Features**

Command to join/leave a group chat

Command to list all the group chats currently active

Text Group Chat Functionality

**Project Structure**

src/main/java: Contains the logic for item registration and dice functionality.

src/main/resources: Holds all 3D models (.json), textures (.png), and sounds.

build.gradle: Manages dependencies and the compilation process.
